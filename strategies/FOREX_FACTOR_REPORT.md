## 四、公式逻辑解读

公式本质是 **"均线差极值反转 + 多层时序平滑"**：

1. `MA_DIFF` — 计算快慢均线差，捕捉短期趋势强度
2. `TS_MIN_10` — 取过去 10 根的最低值
3. `NEG` — 取反
4. `TS_MAX_10` — 取过去 10 根的最高值
5. `TS_MEAN_10 -> TS_MEAN_20 -> TS_MEAN_10 -> TS_MEAN_20` — 多层时序平滑，滤除高频噪声

**经济意义**：当均线差跌到近期低点后反弹（反转信号），叠加多层平滑过滤假信号——典型的均值回归 + 趋势过滤结构，逻辑可解释。

## 五、有效性判定

**✅ 确定有效**，依据：

1. **经济意义明确** — 公式不是符号堆砌（对比 index 组的 `ABS -> ABS -> SIGN -> ABS -> ABS`），每一步都有明确的时序统计含义。
2. **回测稳健** — Sharpe 1.6-1.8，胜率 57%，持仓 2-3 天，是中频策略的健康特征。MaxDD 仅 0.07-0.10，回撤可控。
3. **跨品种一致** — EURUSD 和 USDJPY 共用同一公式，双双盈利，说明因子捕捉的是 forex 类共性而非单品种噪声。
4. **训练分适中** — Best=5.976，非极端高（对比 index 组 7.56 反而过拟合）。因子搜索里，训练分中等往往泛化更好。

## 六、对比：index 组（过拟合失败案例）

| 维度 | forex 组 | index 组 |
|------|----------|----------|
| 公式 | `MA_DIFF -> TS_MIN_10 -> NEG -> ...` | `RET5 -> ABS -> ABS -> SIGN -> ABS -> ABS -> NEG -> TS_RANK_10` |
| 训练 Best | 5.976 | 7.562（更高） |
| 熵坍塌重启 | 0/8 | 8/8（触发上限） |
| 回测交易数 | 83 / 77 | 3 / 3 |
| 回测 Sharpe | +1.82 / +1.63 | -1.35 / -1.60 |
| 判定 | ✅ 有效 | ❌ 过拟合废公式 |

**结论**：训练分高 ≠ 好因子。index 组连续 6 个 ABS/SIGN 导致搜索空间塌缩，训练分虚高但回测崩溃。

## 七、使用建议

- **可小仓位试单**：EURUSD/USDJPY H1 周期，按公式信号方向开仓
- **实盘前必做**：用真实 MT5 点差重测（本次回测用默认点差，实盘可能更宽）
- **持续监控**：若连续 20 笔交易胜率跌破 45%，考虑重新训练
- **不要直接用于 index 品种**：US30/US100/US500 等需要重新训练（避免 index 组的过拟合）
