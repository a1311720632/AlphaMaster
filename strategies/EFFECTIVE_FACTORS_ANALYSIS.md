## 二、Forex 组公式原理分析（确定有效）

### 公式
```
MA_DIFF -> TS_MIN_10 -> NEG -> TS_MAX_10 -> TS_MEAN_10 -> TS_MEAN_20 -> TS_MEAN_10 -> TS_MEAN_20
```
Token: `[3, 45, 24, 44, 33, 34, 33, 34]`

### 栈式虚拟机执行过程

全部为 1 元算子，依次变换：

| 步骤 | 操作 | 栈顶值 | 含义 |
|------|------|--------|------|
| 1 | 压入 MA_DIFF | MA_DIFF | 快慢均线差 = MA10/MA30 - 1 |
| 2 | TS_MIN_10 | TS_MIN_10(MA_DIFF) | 过去10根的最低均线差 |
| 3 | NEG | -TS_MIN_10(MA_DIFF) | 取反 → 反弹潜力 |
| 4 | TS_MAX_10 | TS_MAX_10(-TS_MIN_10(MA_DIFF)) | 反弹潜力的10根最高值 |
| 5 | TS_MEAN_10 | 平滑①（10期均值） | 去噪 |
| 6 | TS_MEAN_20 | 平滑②（20期均值） | 进一步去噪 |
| 7 | TS_MEAN_10 | 平滑③（10期均值） | 三重平滑 |
| 8 | TS_MEAN_20 | 平滑④（20期均值） | 四重平滑 → 最终信号 |

### 逐步逻辑解读

**第 1 步 — MA_DIFF（趋势感知）**
MA_DIFF = MA10/MA30 - 1，衡量短期均线相对长期均线的偏离。正值=多头排列，负值=空头排列。这是趋势的"温度计"。

**第 2 步 — TS_MIN_10（找极值）**
取过去 10 根 K 线的最低 MA_DIFF。当这个值很低（深度负值），说明近期出现过明显的空头趋势——价格在短期内大幅下跌。

**第 3 步 — NEG（反转信号）**
取反。`-TS_MIN_10(MA_DIFF)` 的逻辑是：**近期跌得越狠，反转潜力越大**。这是经典的均值回归假设——价格偏离越远，回弹力度越强。

**第 4 步 — TS_MAX_10（确认反转）**
取反转潜力的 10 根最高值。这一步是"确认机制"：不仅要求反弹潜力存在，还要求它在近期达到过峰值——即反转已经开始发生，而不是还在继续下跌。这避免了"抄底抄在半山腰"。

**第 5-8 步 — 四重交替平滑（去噪）**
TS_MEAN_10 → TS_MEAN_20 → TS_MEAN_10 → TS_MEAN_20，交替使用 10 期和 20 期均值。这种级联平滑等价于一个低通滤波器：
- 第一层 MEAN_10 滤除 10 根以内的噪声
- 第二层 MEAN_20 滤除 20 根以内的波动
- 第三、四层进一步压制残余噪声
- 最终信号非常平滑，只有明确的趋势性信号才能通过

### 经济学原理

**本质：超跌反弹 + 趋势确认 + 多重平滑**

这个公式捕捉的是外汇市场的一个经典现象：**短期超卖后的均值回归**。EURUSD 和 USDJPY 作为主要货币对，其价格行为有强烈的均值回归特性（与股票指数的趋势性不同）——短期内过度偏离后，往往会出现反向修正。

公式的精妙之处在于：
1. **不直接抄底**：先等"反弹潜力"出现（TS_MIN_10 + NEG）
2. **确认反转**：再等反弹潜力达到峰值（TS_MAX_10），避免接飞刀
3. **四重平滑**：最后用四层均值过滤掉所有短期噪声，只保留高置信度信号

这解释了回测特征：
- **持仓 2-3 天**（57-62h）：平滑后的信号变化缓慢，持仓周期中等
- **胜率 54-58%**：不是高频赌博，是中等胜率+正期望
- **MaxDD 仅 0.06-0.10**：四重平滑使信号极度平滑，避免了大幅回撤
- **跨品种一致**：EURUSD 和 USDJPY 共用公式都盈利，说明捕捉的是外汇共性

## 三、Metals_comm 组公式原理分析（不稳定）

### 公式
```
RET20 -> RET -> WMA -> VOL_RATIO -> ATR -> TS_MEAN_20 -> GATE -> TS_CORR_10
```
Token: `[2, 0, 46, 14, 5, 34, 27, 41]`

### 栈式虚拟机执行过程

包含 2 个特征 + 1 个 1 元算子 + 2 个特征 + 1 个 1 元算子 + 1 个 3 元算子 + 1 个 2 元算子：

| 步骤 | 操作 | 栈状态（底→顶） | 含义 |
|------|------|------------------|------|
| 1 | 压入 RET20 | [RET20] | 20期动量 |
| 2 | 压入 RET | [RET20, RET] | 单期收益率 |
| 3 | WMA(RET) | [RET20, WMA(RET)] | 加权平滑收益率 |
| 4 | 压入 VOL_RATIO | [RET20, WMA(RET), VOL_RATIO] | 量比 |
| 5 | 压入 ATR | [RET20, WMA(RET), VOL_RATIO, ATR] | 波动率 |
| 6 | TS_MEAN_20(ATR) | [RET20, WMA(RET), VOL_RATIO, TS_MEAN_20(ATR)] | 波动率均值 |
| 7 | GATE | [RET20, GATE(WMA(RET), VOL_RATIO, TS_MEAN_20(ATR))] | 条件选择 |
| 8 | TS_CORR_10 | [TS_CORR_10(RET20, GATE(...))] | 滚动相关系数 |

### GATE 算子详解

```
GATE(condition, x, y) = (condition > 0) * x + (condition <= 0) * y
```

在第 7 步中，弹出 3 个值（逆序弹出再 reverse）：
- **condition** = WMA(RET)（加权短期收益率）
- **x** = VOL_RATIO（量比）
- **y** = TS_MEAN_20(ATR)（波动率均值）

含义：
- **当 WMA(RET) > 0（价格上涨时）**：信号 = VOL_RATIO → 关注成交量放大程度
- **当 WMA(RET) ≤ 0（价格下跌时）**：信号 = TS_MEAN_20(ATR) → 关注波动率水平

### TS_CORR_10 详解

第 8 步弹出 2 个值：
- **a** = RET20（中期动量）
- **b** = GATE(...)（上述条件信号）

计算 a 和 b 的 10 期滚动 Pearson 相关系数，值域 [-1, 1]。

### 经济学原理

**本质：条件性量价确认策略**

这个公式的逻辑是：
1. **上涨时看量**：当短期收益为正，提取"量比"信号——如果成交量放大（VOL_RATIO 高），说明上涨有量能配合，趋势可信
2. **下跌时看波动**：当短期收益为负，提取"波动率"信号——如果波动率高，可能是恐慌性抛售（可能见底）；如果波动率低，可能是温和回调（可能继续跌）
3. **与中期动量相关**：最后计算这个条件信号与 20 期动量的相关性——正相关说明中期趋势得到了量/波的确认

### 为什么不稳定？

这个公式在 COCOA.c 上 Sharpe 2.54，但在 AAVUSD 上 Sharpe -0.28：

1. **COCOA.c 的特殊性**：可可期货是低流动性品种，量价关系更明显——放量上涨通常意味着真实的供需变化，信号质量高
2. **AAVUSD 的成交量噪声**：AAVUSD（白银相关）的成交量数据质量可能较差，VOL_RATIO 信号失真，导致 GATE 在上涨时选了一个噪声信号
3. **高频交易的成本侵蚀**：1088 笔交易、持仓 7.7h，每笔被点差吃一次——AAVUSD 的点差可能比 COCOA.c 更宽，高频策略在宽点差下必亏
4. **XAUUSD 的 NaN**：黄金数据含 NaN（可能是缓存损坏），无法评估

**结论**：metals_comm 组公式对品种的成交量数据质量高度敏感，不具备跨品种泛化能力。COCOA.c 上的好成绩可能是特例，而非真正的有效因子。

## 四、Index 组公式分析（过拟合失败）

### 公式
```
RET5 -> ABS -> ABS -> SIGN -> ABS -> ABS -> NEG -> TS_RANK_10
```

### 为什么失败

连续 6 个 ABS/SIGN 操作：
- `ABS(ABS(x))` = `ABS(x)` — 冗余
- `SIGN(ABS(x))` = `SIGN(ABS(x))` = 1（当 x≠0 时恒为 1）
- 后续 ABS 对常数 1 无影响
- `NEG(1)` = -1（常数）
- `TS_RANK_10(-1)` = 0（常数在 10 期窗口内排名恒为 0）

**这个公式最终输出接近常数，导致几乎不产生交易信号**（3-5 笔交易，持仓 66-116 天）。训练时触发 8/8 次熵坍塌重启，说明搜索空间塌缩——模型找到了一个"在训练集上得分高但无实际意义"的公式。

训练分 Best=7.562 是三组最高，但回测全亏——**典型的过拟合：训练分越高，泛化越差**。

## 五、总结

| 公式 | 原理 | 有效性 | 根本原因 |
|------|------|--------|----------|
| forex | 超跌反弹 + 趋势确认 + 四重平滑 | ✅ 有效 | 捕捉外汇市场均值回归特性，逻辑稳健，跨品种一致 |
| metals_comm | 条件性量价确认 | ⚠️ 不稳定 | 依赖成交量数据质量，对品种敏感，高频成本侵蚀 |
| index | 符号堆砌（退化） | ❌ 无效 | 连续 ABS/SIGN 导致输出近似常数，过拟合 |

**核心发现**：
1. **好的因子逻辑可解释**：forex 组每一步都有明确的统计含义，对应"超跌反弹"这一经典金融现象
2. **训练分高 ≠ 好因子**：index 组训练分最高(7.56)但回测崩溃，forex 组训练分中等(5.98)但回测最好
3. **跨品种一致性是关键验证**：forex 组在 EURUSD 和 USDJPY 上都盈利，证明因子捕捉的是共性而非噪声
4. **高频策略对成本极敏感**：metals_comm 组 1000+ 笔交易，在宽点差品种上必亏
